# E-Commerce Customer Segmentation using K-Means Clustering

## Overview
A complete end-to-end project to segment e-commerce customers based on purchasing behavior using RFM analysis and K-Means clustering.

## Structure
- `src/`: Core logic modules
- `data/`: Dataset storage (processed data saved here)
- `plots/`: Generated visualizations

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the pipeline: `python main.py`

## Features
- **RFM Analysis**: Recency, Frequency, Monetary value calculation.
- **Clustering**: K-Means with Elbow Method and Silhouette Score for optimal K.
- **Visualization**: 2D/3D cluster plots and detailed segment analysis.
